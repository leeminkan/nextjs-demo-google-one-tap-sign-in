## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## If the Google Popup does not display

Go to chrome://settings/content/federatedIdentityApi on your Chrome Browser, then allow to show third-party sign-in prompts!
